Food Delivery Time Prediction
 Overview

This project predicts food delivery time using machine learning techniques. 
It includes data preprocessing, exploratory data analysis (EDA), feature engineering, and model evaluation using Linear Regression and Logistic Regression.

Objective :
Predict delivery time (regression)
Classify deliveries as Fast or Delayed (classification)

Workflow : 
Data cleaning & preprocessing
One-hot encoding of categorical variables
Feature scaling (Standardization)
Feature engineering (Rush_Hour feature)
Model training (80/20 split)
Performance evaluation

Models Used :
-Linear Regression

-Logistic Regressio

Results indicate weak predictive relationships in the dataset.

Tech Stack :
Python
Pandas
NumPy
Matplotlib & Seaborn
Scikit-learn

Conclusion:
This project demonstrates a complete end-to-end machine learning pipeline and highlights the importance of high-quality data in predictive modeling.
