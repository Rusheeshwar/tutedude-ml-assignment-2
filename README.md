# Global Pollution Analysis

This project analyzes global pollution data and applies machine learning models to study the relationship between pollution levels and energy recovery.

## Objectives
- Perform exploratory data analysis on pollution indicators.
- Predict **Energy Recovered (GWh)** using Linear Regression.
- Classify pollution severity (**Low, Medium, High**) using Logistic Regression.

## Methods
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Linear Regression (prediction)
- Logistic Regression (classification)

## Results
- Linear Regression showed weak predictive performance (low R²).
- Logistic Regression achieved **~90% accuracy** in classifying pollution severity.

## Technologies
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
